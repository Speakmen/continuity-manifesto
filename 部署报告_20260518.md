## 三、部署方案

### 方案A：使用已生成的CAR文件（最简单）

已生成CAR文件：`continuity_manifesto.car`  
CID：`bafybeiabo2w7ybb3jtrhow72tkbukjjb7figyufvtqn7rmccv6omhf2qgq`

#### 方式1：Pinata.cloud（推荐，免费）
1. 注册 https://pinata.cloud
2. 获取 API Key 和 Secret
3. 上传CAR文件：
   ```bash
   curl -X POST "https://api.pinata.cloud/pinning/pinFileToIPFS" \
     -H "pinata_api_key: YOUR_API_KEY" \
     -H "pinata_secret_api_key: YOUR_SECRET" \
     -F "file=@continuity_manifesto.car"
   ```
4. 验证访问：`https://ipfs.io/ipfs/bafybeiabo2w7ybb3jtrhow72tkbukjjb7figyufvtqn7rmccv6omhf2qgq`

#### 方式2：使用ipfs-deploy
1. 注册Pinata获取API密钥
2. 配置环境变量：
   ```bash
   export IPFS_DEPLOY_PINATA_API_KEY="your_key"
   export IPFS_DEPLOY_PINATA_SECRET_API_KEY="your_secret"
   ```
3. 部署：
   ```bash
   npx ipfs-deploy . -p pinata
   ```

### 方案B：web3.storage / storacha（免费）

1. 注册 https://storacha.network
